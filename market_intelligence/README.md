# Quivrr Market Intelligence

This folder contains historical inventory analysis, retailer delta detection, trend reporting and future market intelligence tooling.

Primary runtime areas:

market_intelligence/sql
Database bootstrap and migration scripts.

market_intelligence/retailer_deltas
Daily retailer inventory snapshot and delta detection logic.

market_intelligence/reporting
Daily, weekly and monthly report generation and email delivery.

market_intelligence/analytics
Trend, heatmap and market movement analysis.

market_intelligence/output
Local generated report output. Do not commit generated files unless explicitly required.
